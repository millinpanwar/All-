About

CSV Toolkit is an original, open-source implementation of CSV utilities. It is not affiliated with or based on the code/content of any specific third-party site. You are free to customize the design, add categories, create new tools, and update page copy for SEO.

All conversions run locally in your browser using well-known, permissively licensed libraries (loaded from CDNs): Papa Parse for CSV parsing and SheetJS for Excel reading.
