Video Compressor (in‑browser)

Compress MP4/MOV/WEBM to MP4 using H.264 (ffmpeg.wasm).

Runs fully in-browser using ffmpeg.wasm. Heavy CPU/RAM. For large files use a desktop ffmpeg if this is slow.
